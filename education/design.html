Design Training Introduction

This is the introduction to Design, the part where the overall look and user interface of the app is created.

Design Session 1

This is the intro to graphic design workshop by Global App Initiative.

Design Session 2 - Wireframing

This is the second design workshop for building mobile applications. In this lecture, the wireframing tool Balsamiq is introduced.

Design Session 3 - Illustrator

Students develop more of the basic skills in using Illustrator as a vectorgraphics program, specifically learning the techniques of how to use the pen tool to recreate shapes and common logos.

Design Session 4 - Illustrator

This is the fourth design training session, developing the fundamentals of Illustrator techniques and uses of the pen tool, boolean operations, and other vector fundamentals.

Design Session 5 - Photoshop Intro

This is the first design lecture on photoshop, aimed at teaching designers the necessary skills to use photoshop as part of the workflow for making a good user experience on mobile platforms.

Design Session 6 - Color & Type

This lecture covers the basics of color theory and color balancing schemes, as well as introduces concepts of typography, covers the difference between Serif and San-serif typefaces, and covers how to use different typefaces correctly.
